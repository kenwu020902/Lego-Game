# 🎮 Shape Seeker - 3D Puzzle Game

A strategic layer-by-layer shape sorting game built in Unity. Rotate, peek, and collect shapes to complete each level as fast as possible!

![Game Screenshot](https://via.placeholder.com/800x400?text=Shape+Seeker+Gameplay)

## ✨ Features

- **360° Object Rotation** - Inspect every angle of the 3D object
- **Multi-Layer System** - Shapes are stacked in layers (outer → middle → inner)
- **Long Press to Peek** - See what's underneath before committing
- **5 Extra Loops** - Temporary storage for strategic play
- **Auto-Collect Mechanic** - Stored shapes automatically collect when bins match
- **Star Rating System** - 5 stars for speedrunning, 1 star for casual play
- **Multiple Levels** - Increasing difficulty with complex shape arrangements

## 🎯 How to Play

1. **Rotate** - Click and drag to rotate the object 360°
2. **Collect** - Click shapes that match your current bins (shown at bottom)
3. **Peek** - Long press any shape to see what's underneath
4. **Use Loops** - When stuck, use extra loops to remove shapes temporarily
5. **Auto-Collect** - Shapes in loops automatically go to bins when they match
6. **Complete** - Remove all shapes to finish the level

## 🧠 Design Philosophy

Shape Seeker combines memory, strategy, and speed. The layer system forces players to think ahead - do you remove a shape now using an extra loop, or wait for the bin to change? The peek mechanic adds depth without complexity, making it easy to learn but hard to master.
